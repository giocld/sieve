# Sieve | NBA Analytics

## Overview
Sieve is an analytics dashboard that evaluates NBA player contracts relative to on-court performance. By combining salary data with advanced impact metrics (LEBRON), it identifies market inefficienciesâ€”highlighting players who are significantly underpaid or overpaid.
Check out the Code Documentation here: [Link to Sieve Code Documentation](./CODE_DOCUMENTATION.md)
## Key Features
- **Value Analysis**: Automatically calculates a "Value Gap" for every player.
- **Team Efficiency**: Visualizes how well front offices convert payroll into wins.
- **Interactive Dashboard**: Filter by salary, age, and impact to explore the market.
- **Data Integration**: Merges contract data from Basketball Reference with advanced performance stats.
